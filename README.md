# Pet Adoption MVC

An ASP.NET Core MVC application for managing pet adoptions at an animal shelter.
Built as a learning project to practice MVC architecture, Entity Framework Core, 
and the service layer pattern.

## Features
- Adopter management (create, update, deactivate)
- Animal management
- Adoption tracking with status workflow
- Search and filter functionality
- Soft deletes with full audit trails

## Tech Stack
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- C#

## Architecture
- Service layer pattern with interfaces
- Repository-style data access via EF Core
- ViewModels for view data shaping
- SearchFilters for service layer queries
